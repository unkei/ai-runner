# 敵ラッシュ・中ボス・射撃分散 実装プラン

## 目的

敵の密度と正面から押し寄せる圧力を高め、通常ウェーブの合間に複数発で倒す中ボスを出現させる。また、味方全員の発砲が同時に揃わないよう、個体ごとの射撃間隔を決定論的に分散する。

## 現状と変更後

- 現在の通常ウェーブはステージに応じて4〜7体、最大9体で、生成間隔は2.2秒から短縮される。変更後は初期から8体以上、最大18体を横に広い複数列で正面上方に生成し、間隔も短縮する。
- 現在の複数HP敵は最終ステージに一度だけ出るボスだけである。変更後は通常ウェーブ数を状態で数え、4ウェーブごとに中ボスを1体追加する。中ボスはステージに応じた複数HPを持つが、最終ボスより小さく弱い。
- 現在の味方は初回発射と再発射にID由来の小さな差しかなく、人数が多いと斉射に見えやすい。変更後は各味方に発射回数を持たせ、個体IDと発射回数から広めにずらした決定論的なクールダウンを設定する。

## スコープ

### 対象

- 通常敵ウェーブの人数、頻度、正面編成
- 定期的な中ボスの状態、スポーン、HP、得点、描画
- 味方ごとの初回および継続射撃タイミングの分散
- 上記ルールの自動テスト、README説明、デスクトップ/モバイルのブラウザ検証

### 対象外

- 新しい操作方法、武器種、敵弾種、ステージ構成
- 最終ボスの出現条件や接触時の相互消滅ルール
- 敵同士の衝突・経路探索、永続的な難易度設定画面

## 状態モデルと描画

- `GameState.enemyWavesSpawned` を追加し、通常/archerウェーブの生成回数を保持する。4回目ごとの通常ウェーブ生成直後に `midboss` を正面中央へ追加する。
- `Enemy.type` に `midboss` を追加する。中ボスは `7 + stage * 2` HP、通常敵より低速、最終ボスより小さい描画倍率を使う。
- `Ally.shotsFired` を追加する。初期クールダウンと各発射後クールダウンを、IDと発射回数の整数演算から決定し、テストを再現可能にしたまま発砲を分散する。
- 通常ウェーブは中央を基準に最大6列の複数行へ展開し、全体が画面上方から味方へ向かう。中ボスにはHP表示を維持し、通常敵・archer・最終ボスと区別できる色と大きさを与える。

## 決定的テストとブラウザ検証

- 初期通常ウェーブが8体以上生成され、ステージ上昇時に増加し、上限18体を超えない。
- 通常ウェーブの各個体がトラック内に収まり、複数列・複数行の正面編成になる。
- 4ウェーブ未満では中ボスが出ず、4ウェーブ目で1体だけ生成される。
- 中ボスが1ダメージで生存し、HP回数分の命中で倒れ、専用得点が一度だけ加算される。
- 味方の初回クールダウンと再発射クールダウンが複数値に分かれ、発射回数が増える。
- 既存の独立移動、接触、最終ボス、リセットのテストが通る。
- 1280x720で多数の敵が上方正面から複数列で接近し、中ボスの外見とHP表示、分散した味方射撃、Restart、コンソールエラーなしを確認する。
- 390x844で敵とHUDがキャンバス内に収まり、横スクロールがなく、操作ボタンが使えることを確認する。

## 実装手順とコミット境界

1. 状態モデル、敵ウェーブ編成、中ボス生成、射撃クールダウン分散を `src/state.js` に実装し、`src/tests.js` に決定的テストを追加する。コミット: `Implement enemy swarms and midboss cadence`
2. `src/game.js` に中ボス描画を追加し、`README.md` に敵ラッシュ・中ボス・分散射撃のルールを反映する。コミット: `Render and document enemy rushes`
3. 全差分レビューと自動/ブラウザ検証を行い、問題があれば修正を独立コミットにする。

## 受け入れ条件

- ゲーム開始直後から、従来より多い敵が画面上方の正面からまとまって接近する。
- 4ウェーブごとに、複数回の銃撃を受けても生存する中ボスが1体出現する。
- 味方の発砲タイミングが初回だけでなく継続射撃でも複数のタイミングに分かれる。
- 中ボスの残HPが視認でき、通常敵および最終ボスと見分けられる。
- 既存ゲームルールが退行せず、自動テストと指定ブラウザ検証が通る。

## リスクと前提

- 敵数増加は低性能端末の描画負荷を上げるため、1ウェーブ18体を上限とし、ブラウザで継続動作を確認する。
- 「たまに」は再現可能で分かりやすい4ウェーブごとと解釈する。
- 「正面から」は新しい横方向スポーンではなく、既存の遠方上端からプレイヤー側へ進む方向を維持し、横に広い隊列へ変更する意味とする。
